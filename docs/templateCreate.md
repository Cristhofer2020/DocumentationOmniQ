<h3>Our flow is based on the fact that when the agents create their templates they can edit it but when we approve our templates we cannot edit it, we can only visualize it and it will show us that it is pending for an manager user to approve it and then send it.</h3>

<h3>Nuestro flujo se basa en que cuando los agentes crean sus plantillas pueden editarla pero cuando aprobamos nuestras plantillas no podemos editarla, sólo podemos visualizarla y nos mostrará que está pendiente de que un usuario gestor la apruebe y luego la envíe.</h3>
